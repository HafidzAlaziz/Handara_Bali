# Handara Bali Website

This is the official landing page for **Handara Bali**, built with React, Tailwind CSS, and Framer Motion.

## Features
- **Modern & Premium Design**: Matches high-end Figma concepts.
- **Smooth Animations**: Powered by Framer Motion.
- **Responsive Layout**: Works on all devices.
- **Performance Optimized**: Built with Vite and code-splitting.

## Setup Instructions

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## Icons
Uses [Lucide React](https://lucide.dev/) for modern SVG icons.

## Fonts
- **Serif**: Playfair Display (Google Fonts)
- **Sans**: Outfit (Google Fonts)

## Deployment
This project is deployment-ready for **Vercel** or **Netlify**. Simply connect your repository and set the build command to `npm run build` and output directory to `dist/`.
